# AI Blog & Email Generator using Gemini API

## 📌 Overview

This project is built using Python, Google Colab, and Google's Gemini API. It generates AI-powered blogs and professional emails based on user inputs.

## ✨ Features

- Generate blogs using:
  - Topic
  - Tone
  - Word Count

- Generate emails using:
  - Recipient
  - Purpose
  - Tone

- Uses Google Colab Secrets for secure API key management.
- Blog generation temperature: **0.7**
- Email generation temperature: **0.3**
- Saves generated outputs as:
  - `blog_output.txt`
  - `email_output.txt`

## 🛠 Technologies Used

- Python
- Google Colab
- Google Gemini API

## 📂 Project Files

- `Blog_Email_Generator.ipynb`
- `blog_output.txt`
- `email_output.txt`
